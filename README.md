# Development
基于Unity，FairyGUI，XLua和VRF开发，目前开放全lua脚本和API。

# Features
1.实时搭配饰品，省去敲kv和预载入的查找，比官方的要快得多。

2.预览动作，animationSequence(以前没卵用，v蛇换模型编辑器后就有点用了)。

3.批量编译贴图。

4.图片和音效事件快速查找。

5.读取多语言的相关资源。


# How To Use
